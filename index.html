SlackOff

Rails / React + Redux Live Site Github

SlackOff is a live chat application inspired by Slack

The application combines TCP-based websocket protocol for bi-directional communication and utilization of the virtual DOM update with React.js for its live chat functionality. The Redux architecture for uni-directional data flow streamlines the development cycle. The backend JSON API is constructed with Ruby on Rails.

  • Employed bi-directional communication through Pusher Websocket protocol service.
  • Enabled performant view layer by React framework’s virtual DOM updates.
  • Simplified the development workflow with Redux unidirectional data architecture.
  • Implemented RESTful JSON API on Ruby on Rails backend for streamlined data flow.
